
---------------------------------------------------INSTRUCTIONS----------------------------------------------------
PREREQUISITES: 
	1)Scapy 
	2)Two seperate NICs  (can be used with only one NIC, but with reduced functionality)




STEP 1: Enable monitor mode in one of your NICs			 (this will be your ATTACKING INTERFACE)




STEP 2: Connect to the targeted AP's wifi with the second NIC if present 	  (this will be your MONITORING INTERFACE)	
	
	
	
	
STEP 3: Edit the config.conf file  
	
	Replace the existing values in the file with your own.
	
	
	Double check them for any misspellings.
	
	
	
	
STEP 4: Run fuzz.py
	
		Use sudo python fuzz.py
